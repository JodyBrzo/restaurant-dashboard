## User Story

```
AS often as I get hungry
I WANT to find restaurants near me 
SO THAT I can decide where I want to eat
```

## Acceptance Criteria

```
GIVEN a restaurant dashboard with form inputs
WHEN I search for a zip code and radius
THEN I am presented with a list of restaurants within that radius
WHEN I view a restaurant of interest
THEN I am presented with the restaurant name, the address street name, phone, rating, review count, price and the URL
WHEN I click the restaurant address
THEN I am presented with a google map of the restaurant location and a button to go back

----BONUS----
WHEN I view the restaurant
THEN I am presented with the option to save as a favorite
WHEN I click on favorites
THEN I am presented with a list of restaurants from local storage

```
